About

The engineer behind William Tucker Solutions.

William Tucker

Senior Software Engineer · Legacy Modernization · AI Consultant

I've spent over 12 years building and modernizing enterprise systems at BC post-secondary institutions — Vancouver Island University (2016–present) and Thompson Rivers University (2012–2016). That's Oracle databases, SSO and SAML, REST and SOAP integrations, ETL pipelines, and the kind of legacy Classic ASP and .NET Framework codebases that most organizations quietly depend on.

I serve on the Finance team at VIU, where I've seen firsthand how the right technology can transform operations — and how the wrong approach wastes time and money. That vantage point is what drives the AI consulting side of William Tucker Solutions: practical, no-hype guidance grounded in real engineering and real finance workflows.

What changed everything in the last two years is AI-augmented development. I use AI agents (Claude and GPT) to accelerate delivery by roughly 10x. A Classic ASP system I'd previously have quoted in months, I now modernize to .NET 8 Blazor in days — with full tests and CI/CD. I built dogmap.ca, a production PWA, as a solo side project using the same approach.

Before software, I was a journeyman marine technician (BCIT). I know what it means to build things that have to work in the real world, and I bring that same mindset to every project. I spoke at the BCNET Conference in 2015 on institutional reporting with Groovy Grails, and I care about documentation and knowledge transfer because I've seen what happens without them.

What I Bring to the Table

Legacy Modernization

Classic ASP, .NET Framework, VB6, Oracle PL/SQL → modern .NET 8 Blazor. AI-augmented workflow ships in days what traditional firms quote in months.

Oracle & SQL

Oracle schema design, PL/SQL, T-SQL, PostgreSQL, stored procedures, ETL (Talend). Replaced a 705,714-line PL/SQL package with service-based C# in 8 days.

Security-First Engineering

SSO/ADFS, SAML, OAuth (Google, Apple), JWT, Row Level Security, CORS architecture, role-based access, rate limiting. Built secure by default, not bolted on after.

Full-Stack Delivery

C#/.NET 8, Blazor Server, ASP.NET Core, Next.js, React, TypeScript, Tailwind. Backend to frontend, I ship the whole stack.

DevOps & CI/CD

Git workflows across 30+ repositories, GitLab CI/CD pipelines, health-check-gated deploys, multi-environment release, rollback automation. Infrastructure that lets teams move fast safely.

AI & LLMs

Claude API (direct SDK integration in production), Claude Code with custom hooks and shared team config, and a custom MCP server authored in TypeScript that extends Claude with multi-database Oracle tools. Prompt engineering, context-document design, AI-agent-driven development. Consulting grounded in how I actually work — not repackaged thought leadership.

Reporting

QuestPDF, Crystal Reports, PL/SQL and T-SQL reporting. Replaced legacy licensed Crystal Reports with .NET-native QuestPDF to eliminate licensing costs.

Team Leadership

Mentored developers, authored team handbooks, established feature-branch and CI/CD workflows for teams of six. Code review, Agile/Scrum, knowledge transfer.

Cloud & Integrations

Vercel, Supabase (Auth, DB, Storage), cloud ERP integrations (REST/SOAP), PostGIS geospatial, SignalR real-time, SSH/SFTP file transfer.
